# Compiling COAWST in DelftBlue
Instructions to compile [COAWST](https://github.com/DOI-USGS/COAWST) in the [DelftBlue](https://doc.dhpc.tudelft.nl/delftblue/) supercomputer.

This instructions are specific for [COAWST v3.8](https://github.com/DOI-USGS/COAWST/releases/tag/COAWST_v3.8). These instructions are complementary to the [User Manual](https://github.com/DOI-USGS/COAWST/blob/COAWST_v3.8/COAWST_User_Manual.doc) included in the COAWST repo.

## 1. Download the repository

Start by logging into DelftBlue and cloning the COWAST GitHub repository to your scratch repository:
```bash
cd /scratch/<netid>
git clone https://github.com/DOI-USGS/COAWST.git
cd COAWST
git checkout COAWST_v3.8
```
